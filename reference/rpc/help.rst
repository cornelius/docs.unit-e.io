help
----

``help ( "command" )``

List all commands, or get help for a specified command.

Argument #1 - command
~~~~~~~~~~~~~~~~~~~~~

**Type:** string, optional

The command to get help on

Result
~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - text
     - string
     - The help text
